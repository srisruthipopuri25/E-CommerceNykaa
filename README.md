# E-CommerceNykaa

Nykaa‑Inspired E‑Commerce Platform

Live Demo: https://e-commerce-nykaa.vercel.app/

A full‑stack e‑commerce application inspired by Nykaa, built with a modern React + Next.js frontend, a Node.js + Express backend, MongoDB for data storage, Redis for caching/recommendations, and Docker for local development.

# Run Project

docker compose up --build

# Features

Product listing and product detail pages

AI‑based product recommendations (Gemini API)

Redis‑based caching and user view history

Responsive UI (mobile, tablet, desktop)

Skeleton loaders for better UX

Dockerized backend services

# Tech Stack

Frontend

Next.js (React)

Tailwind CSS

Axios

Zustand (state management)

Backend

Node.js

Express.js

MongoDB (Atlas)

Redis

Google Gemini API (AI recommendations)
